# symulator-wyprawy-statku
Projekt Python – symulator wyprawy statku po dwuwymiarowym świecie.
Symulator wyprawy statku

Program został napisany w Pythonie i działa w terminalu.

Symulacja przedstawia wyprawę statku po dwuwymiarowym świecie.
Gracz steruje ruchem statku, zarządza energią i próbuje dotrzeć do wyznaczonego celu.

Program zawiera:

współrzędne x i y,
granice świata,
system energii,
losowe zdarzenia,
przeszkody i bonusy,
raport końcowy,
możliwość ponownego uruchomienia symulacji.
Uruchomienie programu

Uruchom plik main.py poleceniem:

python main.py
Wymagania
Python 3.11
brak dodatkowych bibliotek

Program korzysta wyłącznie ze standardowej biblioteki Pythona (random).
